# 📱 SotkaMarket — Агрегатор предложений мобильной электроники

**SotkaMarket** — это современная, производительная PWA (Progressive Web App) витрина для маркетплейса электроники. Платформа консолидирует предложения от различных продавцов, позволяя пользователям подбирать гаджеты, сравнивать цены (как на новые, так и на Б/У устройства) и находить магазины на интерактивной карте.

Проект спроектирован с упором на скорость работы, чистый код (Vanilla JS без тяжелых фреймворков) и безупречный UI/UX-опыт у пользователя на мобильных и десктопных устройствах. Топовый дизайн и гладкие анимации прилагаются.

---

## 🌟 Ключевые возможности

- 🛒 **Глубокий структурированный каталог:** Поддержка классов, категорий, брендов и фильтров (цена, состояние, память, цвет).
- 🔄 **Иерархическая логика данных:** Модель (абстрактный товар) -> SKU-версия (Цвет + Память) -> Конкретное предложение магазина (Offer с конкретной ценой и состоянием).
- 📍 **Интерактивная карта (Leaflet):** Поиск магазинов и офлайн-точек продаж прямо на удобной карте Бишкека.
- 📱 **Progressive Web App (PWA):** Возможность установки приложения на главный экран смартфона (Add to Home Screen) и базовая работа офлайн благодаря Service Worker.
- ⚡ **Ультра-быстрый клиентский роутинг:** Мгновенный переход между страницами на основе Hash-роутера (без перезагрузки страниц).
- 🎨 **Адаптивный дизайн (Tailwind CSS):** Идеальное отображение на смартфонах, планшетах и мониторах высокой четкости. Плавные микро-взаимодействия и анимации.
- 💖 **Избранное:** Сохранение любимых моделей и текущих предложений в `localStorage`.

## 🛠 Технологический стек

-   **Язык разметки и стили:** HTML5, CSS3, **Tailwind CSS** (через CDN для скорости развертывания прототипа).
-   **Логика:** Vanilla JavaScript (ES6+), модульная система (ES Modules).
-   **Маршрутизация:** Самописный SPA-роутер на базе `window.onhashchange`.
-   **Стейт-менеджмент:** Глобальный реактивный объект `window.State`, синхронизируемый с `localStorage`.
-   **Картография:** **Leaflet.js** + OpenStreetMap.
-   **Иконки:** Встроенная кастомная SVG-графика.
-   **Шрифты:** Nunito (Google Fonts).

## 🏃 Запуск проекта локально

Поскольку проект не использует Node.js для сборки (Tailwind работает через CDN, а JS исполняется нативными модулями браузера), запуск крайне прост:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/kutya001/SMarket.git
    cd SMarket
    ```
2.  **Запустите любой локальный статический сервер:**
    Так как используются ES6-модули (`type="module"`), просто открыть `index.html` через `file://` протокол в браузере не получится из-за CORS-политик браузера.
    
    *   С помощью Python: `python -m http.server 3000`
    *   С помощью Node.js: `npx serve .` или `npx lite-server`
    *   Или установите расширение **Live Server** для VS Code.
3.  **Откройте приложение:** Перейдите по адресу `http://localhost:3000` в браузере.

## 🧠 Устройство логики (Data Flow)

Текущий прототип работает на базе мок-данных (`mockData.js`). 

Приложение строится вокруг объекта **`window.State`**:
```javascript
export const State = {
  favorites: [...],
  user: null,
  filters: { class: [], category: [], brand: [], seller: [], priceMin: 0, priceMax: 999999, condition: 'all', ... },
  sort: 'popular',
  catalogView: 'products', // products | offers | sellers | brands
};
```
События из UI вызывает глобальные методы, привязанные прямо к объекту `window`, например `window.updateFilter()`, `window.addToFavorites()`, после чего вызывается функция `renderCurrentPage()`. Компоненты страниц перехватывают управление, читают текущий State, генерируют HTML-строки с контентом и вставляют их в корневой контейнер `main#app`.
